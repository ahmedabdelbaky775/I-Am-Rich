# I Am Rich 💎

This is my first iOS application developed as part of the iOS & Swift Bootcamp.

## Description
The app is a simple single-screen application that displays a diamond image and a label, symbolizing wealth. It was created to practice UI layout and project structure in Xcode.

## What I learned:
* How to use Interface Builder (Storyboard).
* Working with Assets and App Icons.
* Setting up Git and GitHub with Xcode.
